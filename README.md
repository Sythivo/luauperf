# luauperf

Flame graph generator for perf profiles. Ported from the Luau `perfgraph.py` and `perfstat.py` tool.

## Usage

### Graph:
```bash
zune run perf graph <input.out> [output.html]
```

### Stats:
```bash
zune run perf stats <input.out>
```
